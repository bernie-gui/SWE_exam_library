# Examples Index

This file contains a summary of the examples located in this directory.

- **backtracking_test.cpp**: Tests the backtracking utility by finding the argument that minimizes a constant function.
- **example_01_26_1.cpp**: Simulates a system of UAVs moving in 1D space to estimate the collision rate using Monte Carlo simulation.
- **example_01_26_2.cpp**: Simulates a system of UAVs with collision avoidance logic, using Monte Carlo to estimate the collision rate.
- **example_01_26_3.cpp**: Simulates a supply chain with customers, servers, and suppliers to estimate the miss rate of customer requests.
- **example_01_26_4.cpp**: Optimizes the number of suppliers in a supply chain simulation to minimize a cost function balancing supplier count and service miss rate.
- **example_01_26_5.cpp**: Optimizes the supplier velocity parameter to minimize a cost function involving velocity and service miss rate.
- **example_02_25_3.cpp**: Simulates a system where client behavior follows a Markov chain, estimating the service rate.
- **example_03_25_1.cpp**: Simulates vehicles moving in a bounded space and deactivates them upon collision, counting survivors at the end.
- **example_03_25_2.cpp**: Optimizes the velocity parameter of vehicles to maximize the fraction of vehicles that avoid collisions.
- **example_03_25_3.cpp**: Simulates a booking system to estimate the rate of overselling when customer demand exceeds available inventory.
- **example_03_25_4.cpp**: Simulates a booking system to estimate transaction success rates and other metrics.
- **example_03_25_5.cpp**: Optimizes a probability parameter in a booking simulation to maximize a utility function based on transaction outcomes.
- **example_06_25_1.cpp**: Simulates particles moving in 2D with sinusoidal trajectories and logs their positions to a file.
- **example_06_25_2.cpp**: Simulates vehicles in 2D with collision avoidance/detection logic, estimating the collision rate.
- **example_06_25_3.cpp**: A basic simulation skeleton that outputs random request IDs at fixed time intervals.
- **example_06_25_4.cpp**: Simulates a server-dispatcher system handling requests to estimate the queue length or system load.
- **example_06_25_5.cpp**: Optimizes a probability parameter in a server-dispatcher simulation to minimize the system load.
- **example_07_25_1.cpp**: Simulates vehicles navigating to targets while avoiding collisions, estimating the collision rate.
- **example_07_25_2.cpp**: Optimizes the navigation weight parameter of vehicles to minimize the collision rate.
- **example_07_25_3.cpp**: Simulates a customer-server system with restocking logic to estimate the request miss rate.
- **example_07_25_4.cpp**: Simulates a customer-server system to estimate both the miss rate and the total number of processed requests.
- **example_07_25_5.cpp**: Optimizes the restocking probability in a customer-server simulation to maximize system throughput.
- **example_09_01_1.cpp**: Simulates a Markov Decision Process (MDP) representing software development steps to calculate the expected accumulated cost.
- **example_09_01_2.cpp**: Simulates an MDP to estimate the probability that the total cost remains within a specified budget.
- **example_09_25_1.cpp**: Simulates a swarm of UAVs with position-dependent velocity changes to estimate the collision rate in a bounded space.
- **example_09_25_2.cpp**: Simulates UAVs using a predictive policy with position caching to minimize a proximity-based objective function.
- **example_09_25_3.cpp**: Simulates a shop system with customers, servers, and suppliers to estimate the rate of missed sales.
- **example_09_25_4.cpp**: Optimizes the number of suppliers in a shop simulation to minimize a combined cost of suppliers and missed sales.
- **example_11_25_1.cpp**: Simulates a Markov chain to calculate the accumulated cost of transitions.
- **example_11_25_2.cpp**: Simulates a Markov chain to estimate the probability that the total cost stays within a maximum limit.
- **example_11_25_3.cpp**: Optimizes a transition probability in a Markov chain simulation to minimize a specific cost function.
- **example_11_25_4.cpp**: Simulates a customer-server system with probabilistic restocking to estimate the request miss rate.
- **example_11_25_5.cpp**: Optimizes the restocking probability in a customer-server simulation to minimize a combined cost of restocking and missed sales.
